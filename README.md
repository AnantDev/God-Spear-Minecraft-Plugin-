# God-Spear-Minecraft-Plugin-
God Spear is a production-ready Minecraft plugin that introduces a unique progression-based spear system. Every player receives a personal Special Spear that evolves through combat, upgrading from Wooden → Stone → Iron → Diamond → Netherite → God Spear by defeating other players with the spear's final killing blow.
Built with performance, security, and scalability in mind, the plugin is designed for modern Minecraft servers running Paper, Purpur, and Folia, with official support for Minecraft 1.21.11 and a future-ready architecture for newer versions.
✨ Features
🗡️ Unique personal Special Spear for every player
📈 Progression system based on valid PvP kills
⚔️ Upgrades through six evolution stages
👑 Final God Spear with Sharpness VII, Lunge V, and Unbreakable
🔒 Persistent ownership using UUID and Persistent Data Containers (PDC)
🚫 Advanced anti-duplication and anti-exploit protection
🛡️ Anti kill-farming system with configurable safeguards
⚡ Optimized for large multiplayer servers
📦 SQLite, MySQL, and MariaDB support
⚙️ Fully configurable through YAML configuration files
🌍 Native Paper, Purpur, and Folia compatibility
🚀 Future-ready architecture for upcoming Minecraft releases
🎯 Goal
Deliver a polished, secure, and highly optimized progression weapon system that provides long-term gameplay while maintaining fairness, stability, and excellent server performance.
